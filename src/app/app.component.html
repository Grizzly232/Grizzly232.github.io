2nd Year Project

I enjoyed the teamwork and collaboration, especially with Github to manage our work.

The final project was a full-fledged online shop with a user profile system, forum, review section, cart, checkout. The project showed me about recording and storing user data properly and gave me plenty of experience with different database management systems. I also used Bootstrap extensively.
Voice-Activated Lighting
I used a Raspberry PI 3 and the Google Speech-to-Text API to activate and control a set of LED lights. This was my first side project where I used an API and also my first time playing with a Raspberry PI. After a while working on college projects with a limited number of dependencies, it was interesting and insightful to work on a project of my own with APIs that I chose myself.

Angular App

I also gained plenty of experience with Jenkins in this project by maintaining two build pipelines.

In R, along with two other students, I analyzed a dataset on the topic of relay races and demonstrated which attributes of an athlete were more important. The project introduced me to R and data analytics, both of which I used later.
